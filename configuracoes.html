Modo Demonstração Ativo
O sistema está usando dados de exemplo locais. Para conectar ao Google Sheets, insira a URL do seu Web App publicado abaixo.
Cole aqui a URL gerada ao publicar o Apps Script como Web App.
Guia de Implantação
1
Criar a Planilha Google Sheets
Acesse Google Sheets → crie uma nova planilha → copie o ID da URL (entre /d/ e /edit).
2
Abrir o Apps Script
Na planilha, vá em Extensões → Apps Script. Cole o conteúdo do arquivo code.gs no editor.
3
Executar a função setup()
No Apps Script, selecione a função setup e clique em Executar. Autorize as permissões solicitadas. Isso criará todas as abas automaticamente.
4
Publicar como Web App
Clique em Implantar → Nova implantação → Tipo: Web App. Configure: "Executar como: Eu" e "Quem tem acesso: Qualquer pessoa". Copie a URL gerada.
5
Configurar no sistema
Cole a URL copiada no campo acima e clique em Salvar. Teste a conexão para verificar.
6
Publicar no Vercel
Suba os arquivos para um repositório GitHub e conecte ao Vercel. O sistema estará online em minutos.
